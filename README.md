# 🍱 Lunch Bot — бот для сверки оплат обедов

## Быстрый старт

### 1. Создать бота в Telegram
1. Напиши [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot` и следуй инструкциям
3. Скопируй токен (выглядит как `123456:ABC-DEF...`)

### 2. Узнать свой Telegram ID
1. Напиши [@userinfobot](https://t.me/userinfobot)
2. Скопируй цифровой ID (это твой `ADMIN_ID`)

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Задать переменные окружения
```bash
export BOT_TOKEN="твой_токен_от_BotFather"
export ADMIN_ID="твой_telegram_id"
```

Или создай файл `.env` и используй `python-dotenv`:
```
BOT_TOKEN=123456:ABC-DEF...
ADMIN_ID=987654321
```

### 5. Запустить бота
```bash
python bot.py
```

## Команды

### Офис-менеджер (Admin)
| Команда | Описание |
|---|---|
| `/setphone +77001234567` | Задать номер для переводов |
| `/setmenu` | Задать меню дня |
| `/report` | Отчёт: кто оплатил, кто нет |
| `/remindall` | Напомнить всем должникам |
| `/closeday` | Итог дня |

### Сотрудники
| Команда | Описание |
|---|---|
| `/lunch` | Выбрать обед из меню |
| `/paid` | Получить реквизиты для оплаты |
| _(отправить фото)_ | Отправить чек — уходит менеджеру |

---

## Как работает флоу

```
Утро:      Менеджер → /setmenu → публикует меню
Обед:      Сотрудник → /lunch → выбирает блюдо → переводит деньги → фото чека
           Бот автоматически пересылает чек менеджеру с кнопками ✅/❌
Вечер:     Менеджер → /report → видит кто оплатил / кто нет
           /remindall — бот рассылает напоминания должникам
```

---

## Статусы заказа
- `ordered` — взял обед, не оплатил
- `pending_confirm` — отправил чек, ожидает подтверждения  
- `confirmed` — оплата подтверждена ✅

## Деплой на сервер (опционально)

Для постоянной работы можно использовать `screen`, `tmux` или `systemd`:

```bash
# screen
screen -S lunchbot
python bot.py
# Ctrl+A, D — отсоединиться
```

Или создай systemd-сервис для автозапуска.
